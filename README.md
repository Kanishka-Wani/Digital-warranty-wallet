# 📦 Digital Warranty Wallet

## Overview

Digital Warranty Wallet is a web application that helps users securely organize product purchase details, warranty information, and bill records in one place.

Many people lose purchase receipts or forget warranty expiration dates, making warranty claims difficult. This application solves that problem by providing a centralized dashboard to manage warranties and quickly access important product information.

---

## Problem Statement

Product bills and warranty cards are often misplaced, resulting in:

* Difficulty claiming warranties
* Missed warranty expiration dates
* Poor organization of product records

Digital Warranty Wallet provides a simple solution by storing all warranty-related information digitally and making it easily accessible.

---

## Features

### Product Management

* Add new products
* Edit product information
* Delete products
* View product details

### Warranty Tracking

* Store purchase date
* Store warranty duration
* Automatically calculate warranty expiry date
* Track remaining warranty period

### Bill Management

* Upload bill images
* Preview stored bills
* Quick access during warranty claims

### Dashboard

* Total products count
* Active warranties count
* Expiring soon warranties count
* Expired warranties count

### Search & Filter

* Search products by name
* Filter by warranty status
* Quickly find required products

## User Flow

